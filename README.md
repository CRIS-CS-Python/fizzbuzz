# FizzBuzz

Write a program that
* prints each number up to `N`
* "Fizz" if the number is divisible by 3,
* "Buzz" if it is divisible by 5,
* or "FizzBuzz" if it is divisible by both 3 and 5.

Example:
```
$ py fizzbuzz.py 15
1
2
3 Fizz
4
5 Buzz
6 Fizz
7
8
9 Fizz
10 Buzz
11
12 Fizz
13
14
15 FizzBuzz
```
